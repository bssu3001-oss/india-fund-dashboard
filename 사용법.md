# 인도 펀드 대시보드 사용법

## 매일 이렇게 하면 돼요

1. 터미널(Terminal) 앱 열기
2. 아래 명령어 입력:
   ```
   cd ~/Desktop/ai-work/인도펀드_대시보드
   python3 업데이트.py
   ```
3. 브라우저가 자동으로 열리면서 최신 데이터가 보여요

---

## 설정 변경 (설정.json)

| 항목 | 설명 |
|------|------|
| `매수단가` | 내가 산 펀드 평균 단가 |
| `투자금_만원` | 현재 투자한 금액 |
| `추가투자_만원` | 추가로 투자할 예정 금액 |
| `손절기준가` | 이 가격 아래면 손절 |
| `NIFTY50_매수시점` | 펀드 살 때 NIFTY50 지수 (현재 추정값 계산에 사용) |
| `현재기준가` | 오늘 실제 기준가를 직접 입력하면 더 정확해요 (숫자 입력, 예: 786.34) |
| `anthropic_api_key` | AI 질문 기능 쓰려면 여기 입력 |

### 현재 기준가 직접 입력하는 방법
앱(KB스타뱅킹 등)에서 오늘 펀드 기준가 확인 후:
```json
"현재기준가": 786.34
```
이렇게 숫자를 넣으면 정확하게 계산돼요.
`null`로 두면 NIFTY50 비율로 자동 추정합니다.

## AI 질문 기능 켜기
1. https://console.anthropic.com 에서 API 키 발급
2. `설정.json` 파일에서 `anthropic_api_key` 항목에 키 입력
3. 다시 `업데이트.py` 실행
